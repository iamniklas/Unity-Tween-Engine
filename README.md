# Unity-Tween-Engine

The Unity Tween Engine is a plug-in for the Unity Engine that provides interpolated transitions.

Available interpolations are:
- **Linear**
- **Quad**
- **Cubic**
- **Quart**
- **Quint**
- **Expo**
- **Circ**
- **Back**
- **Elastic**
- **Bounce**

All interpolations provide the 3 types (In, Out, InOut).
